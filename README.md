# Tic-Tac-Toe

A simple Tic-Tac-Toe game built with HTML, CSS, and JavaScript.

## Features
- Interactive game board
- Three difficulty levels: Easy, Hard, Impossible
- Player vs. Bot gameplay
- Score tracking
